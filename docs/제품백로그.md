# 제품 백로그 (Product Backlog)

> 참조: `docs/요구사항분석.md`

## 백로그 개요

배드민턴 매칭 서비스의 제품 백로그입니다. 백엔드(API)와 프론트엔드(웹/앱) 작업을 User Story 중심으로 정리했습니다.

**우선순위**: P0(필수) > P1(중요) > P2(선택)
